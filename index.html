Environment

  • Install Visual Studio Code
  • Install NodeJS + Package Manager

    Node.js is an open source server environment

    • Dowload and Install
    • npm init

      package manager solution init

    • package.json

      Package Manager config file

    • npm run [task]

      run custom stask

    • create source directory [./src]
  • Install Typescript 3

    Javascript superset

    • npm install typescript --save-dev

      local installation - [./node_modules/.bin/tsc]

    • ./node_modules/.bin/tsc --init + tsconfig.json

      init tsc + config file [ "outDir": "./dist", "rootDir": "./src"]

    • create custom task - package.json

      ["tsc": "./node_modules/.bin/tsc"]

    • index.ts

      create index.ts file inside ./src folder -> npm run tsc

  • Install Lite-server

    Lite Web Server

    • npm install lite-server --save-dev

      local installation - [./node_modules/.bin/lite-server]

    • create bs-config.json

      { "port": 8000, "files": ["./dist/**/*.{html,htm,css,js}"], "server": { "baseDir": "./" } }

    • create index.html

      script src="dist/index.js"

    • create custom task - package.json

      ["lite-server" : "./node_modules/.bin/lite-server"]

  • Put it togheter - package.json

    ["start" : "npm run tsc && npm run lite-server"]
